## 快速开始（只需 2 步）

```bash
# 第一步：收集新闻
# 向 AI 发送：
#   "搜索 Hacker News 上 4月7日 到 4月13日 的热点新闻 每天 要有10条"
#
# → AI 返回带编号的列表，你查看后决定哪些需要深度解读

# 第二步：深度解读 + 归档（一站式完成）
# 向 AI 发送（三选一）：
#   A. "对 #1.3、#2.7、#3.5 这几篇做深度解读"     ← 指定编号
#   B. "对认知暗森林、Copilot广告事件做深度解读"     ← 指定标题
#   C. "对所有文章做深度解读"                        ← 不指定 = 全部
#
# → AI 自动完成：撰写解读 → 归档文件 → 生成周度索引 → 更新README → 全项目检查

# （可选）后续清理
# 向 AI 发送：
#   "清理部分文档"
# → 删除 README 中不需要的索引条目和对应文章文件（week.md 列表不动）
```
